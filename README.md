# 🎮 2048 Game (JavaScript)

A simple and clean implementation of the classic **2048 game** built using **HTML, CSS, and JavaScript**. Slide tiles, merge numbers, and try to reach **2048**!

---

## 🚀 Features

* 🎯 Classic 4x4 grid gameplay
* ⌨️ Keyboard controls (Arrow keys)
* 🔢 Score tracking
* 🎨 Dynamic tile colors based on values
* ❌ Game Over detection
* 🔄 Retry button to restart instantly

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling and layout
* **JavaScript** – Game logic

---

## 🎮 How to Play

* Use the **arrow keys**:

  * ⬅️ Left
  * ➡️ Right
  * ⬆️ Up
  * ⬇️ Down

* When two tiles with the same number touch, they **merge into one**

* Each move spawns a new tile (2)

* The game ends when:

  * No empty spaces remain
  * No valid moves are possible

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start playing 🎉

---

## 🔄 Restarting the Game

* When the game ends, a **Game Over screen** appears
* Click the **Retry button** to start a new game
